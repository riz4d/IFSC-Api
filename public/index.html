What's IFSC API ?

IFSC API can be used to retrieve information about a specific bank branch using its IFSC code. This information may include details such as the bank's name, branch name, address, contact information, and more

Usage

Get Method

GET/?ifsc={IFSC-CODE}

Example:

https://ifsc.rizad.me/?ifsc=UCBA0001043
